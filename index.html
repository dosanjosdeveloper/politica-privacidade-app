Política de Privacidade

Aplicativo:Plano ACS

Última atualização:21 de Abril de 2026

1. Visão Geral

O Plano ACS foi desenvolvido para auxiliar na gestão de indicadores de saúde e acompanhamento de pacientes. Este aplicativo foi projetado com foco total na privacidade do usuário, operando de forma offline e local.

2. Coleta e Uso de Dados

Nós não coletamos, não transmitimos e não compartilhamos qualquer informação pessoal ou sensível inserida no aplicativo. Todos os dados de pacientes, categorias e registros de indicadores permanecem exclusivamente sob posse do usuário.

3. Armazenamento Local (SQLite)

As informações são armazenadas em um banco de dados local (SQLite) dentro do diretório privado do aplicativo no dispositivo. Não há sincronização com servidores externos ou armazenamento em nuvem.

4. Autenticação e Acesso

Este aplicativo não possui sistema próprio de autenticação (login/senha). A segurança do acesso aos dados contidos no aplicativo depende exclusivamente das travas de segurança (senha, padrão ou biometria) configuradas pelo usuário no sistema operacional do dispositivo.

5. Exclusão de Dados

O usuário tem controle total sobre suas informações. Ao desinstalar o aplicativo, o sistema Android removerá automaticamente o banco de dados local, resultando na exclusão permanente de todos os registros.
